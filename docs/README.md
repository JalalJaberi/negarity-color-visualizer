# Negarity Color Visualizer — documentation site

This folder is a **Docusaurus 3** site (same template, theme, and static assets as **negarity-color** docs).

## Commands

```bash
npm install
npm start          # dev server
npm run build      # production build (Node >= 20)
npm run serve      # serve the build locally
```

## Deploy notes

- **`docusaurus.config.ts`** — set **`url`** and **`baseUrl`** for your host (default assumes `https://negarity.com/docs/negarity-color-visualizer`).
- Static branding: **`static/img/logo.png`**, **`static/img/favicon.png`** (copied from negarity-color docs; replace if needed).

## Content

Markdown lives in **`docs/docs/`**. The sidebar is **autogenerated** from that tree (`sidebars.ts`).

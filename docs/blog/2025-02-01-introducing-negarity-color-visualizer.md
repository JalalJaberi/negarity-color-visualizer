# Introducing Negarity Color Visualizer

**Negarity Color Visualizer** is a TypeScript library for building **2D** (Konva) and **3D** (Three.js) color-space views in the browser.

## Highlights

- **`ColorVisualizer`** — one entry class for presets, resize, and mode switching between 2D and 3D.
- **Built-in 3D preset** — RGB cube (`RGB_CUBE_PRESET`).
- **Color channel UI** — `ColorChannelVisualizer` with sliders for RGB, CMYK, HSL, HSV, Lab, LCh, XYZ, and YCbCr.
- **Shipped as a bundle** — Three.js and Konva are dependencies of the package; consume via npm and your bundler.

See the [documentation](/docs/intro) for installation and API details.

Channel Sliders

How the Color Channel Visualizer (sliders) work

What is this?

The ColorChannelVisualizer renders one slider per channel of a color space (e.g. R, G, B for RGB or H, S, L for HSL). Each slider shows a gradient and updates a shared color. You can use onChange / onChannelChange to react to changes, and setValues() to update the sliders from code (e.g. presets).

API at a glance

import { ColorChannelVisualizer, valuesToHex } from 'negarity-color-visualizer'; const ccv = ColorChannelVisualizer(containerElement, { colorSpace: 'HSL', // 'RGB' | 'HSL' | 'HSV' | 'CMYK' | 'LAB' | 'LCh' | ... values: { h: 0, s: 100, l: 50 }, showPreview: true, // show color swatch (default true) layout: 'vertical', // 'vertical' | 'horizontal' onChange(values) { console.log('Color:', valuesToHex('HSL', values), values); }, onChannelChange(channelKey, value) { console.log(channelKey, '=', value); }, }); // Update from outside ccv.setValues({ h: 120, s: 80, l: 50 }); // Read current const values = ccv.getValues(); const hex = valuesToHex(ccv.getColorSpace(), values); // Cleanup ccv.destroy();
